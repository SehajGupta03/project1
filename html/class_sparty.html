void Sparty::Draw ( std::shared_ptr< wxGraphicsContext > gc)
overridevirtual

Draws Sparty on the given graphics context.

The drawing is done so that the Sparty is centered on the x and y location given in the XML. The drawing of the boot is animated so that it looks like it is kicking. The animation is currently just a placeholder and the time since the last kick is hardcoded to 0.

Initiates the kick animation for Sparty.
